title=Theoretical Regular Expression Library shortname=RE Library tags=library haskell regexp

This library will be coming soon, and is for working with the theory of regular expressions and state machines. A Haskell library for manipulating regular expressions. It is very different from Text.Regex. It is intended to release this library as open source.

You can view early and incomplete Haddock documentation for the library here. If you are interested in this library then please contact me, particularly if you have any feedback or would like to use the library.

External Tools

In order to generate cool graphs you will need dot installed on your system. It's available from http://www.graphviz.org/. Everything else requires any Haskell compiler/interpreter.
